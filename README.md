# sap-customer-order-alv-report
SAP S/4HANA ortamında VBAK tablosu kullanılarak geliştirilen, müşteri bazlı siparişleri ALV (SALV) ile listeleyen ve müşteri ile tarih aralığına göre dinamik filtreleme sağlayan ABAP raporlama uygulaması.
📌 SAP ABAP – Müşteri Sipariş Listeleme Raporu (ALV)
📖 Proje Açıklaması

Bu proje, SAP S/4HANA ortamında VBAK tablosu kullanılarak geliştirilmiş bir müşteri sipariş listeleme raporudur. Kullanıcıların müşteri numarası ve tarih aralığına göre siparişleri filtreleyerek ALV (SALV) ekranında görüntülemesini sağlar.

🎯 Problem

SAP standartlarında müşteri siparişlerinin hızlı, filtrelenebilir ve kullanıcı dostu şekilde görüntülenmesini sağlayan esnek bir rapor ihtiyacı bulunmaktadır.

🛠️ Çözüm

Bu ihtiyaç doğrultusunda aşağıdaki geliştirmeler yapılmıştır:

VBAK tablosu üzerinden sipariş verilerinin çekilmesi
Müşteri numarası (PARAMETERS) ile filtreleme
Tarih aralığı (SELECT-OPTIONS) ile dinamik sorgulama
ALV (CL_SALV_TABLE) kullanılarak listeleme ekranı oluşturulması
Kolon optimizasyonu ile kullanıcı deneyiminin iyileştirilmesi
Başlık bilgisi ile raporun daha anlaşılır hale getirilmesi
⚙️ Kullanılan Teknolojiler
SAP ABAP
SAP S/4HANA
VBAK (Satış Siparişleri)
ALV (SALV Model)
SELECT-OPTIONS
PARAMETERS
📊 Özellikler

✔ Müşteri bazlı sipariş filtreleme
✔ Tarih aralığına göre sorgulama
✔ ALV tabanlı kullanıcı dostu ekran
✔ Kolon optimizasyonu
✔ SAP standartlarına uygun raporlama
